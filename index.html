📋 JSON 형식 가이드

문제 배열([ ]) 안에 각 문제 객체를 넣어주세요.

① 객관식 type: "multiple"

{
  "type": "multiple",
  "question": "Carbon is considered to be _______.",
  "choices": ["tetravalent", "divalent", "trivalent", "monovalent"],
  "answer": 0,
  "explanation": "탄소는 4개의 공유결합을 형성합니다."
}

② 단답형 type: "short"

{
  "type": "short",
  "question": "물의 화학식은?",
  "answer": "H2O",
  "explanation": "수소 2개 + 산소 1개"
}

③ 서술형 type: "essay"

{
  "type": "essay",
  "question": "공유결합이란 무엇인지 설명하시오.",
  "answer": "두 원자가 전자쌍을 공유하여 형성하는 화학결합",
  "explanation": "이온결합과 달리 전자를 주고받지 않고 공유합니다."
}
  • question 문제 텍스트 (필수)
  • type "multiple" / "short" / "essay"
  • choices 보기 배열 (객관식 필수)
  • answer 객관식: 0-based 인덱스 / 나머지: 정답 문자열
  • explanation 해설 (선택)
  • image 파일명, URL, 또는 base64 (선택)
